# Job Notification App Design System

## Overview
A premium, calm, and intentional design system for a B2C Job Notification App. The system prioritizes clarity, confidence, and authority through a minimal color palette, generous whitespace, and sophisticated typography.

## Core Principles
- **Intentional Whitespace**: Using a strict 8px-based spacing scale.
- **Limited Palette**: Max 4 colors to reduce visual noise.
- **Typography Contrast**: Pairing a confident Serif heading with a clean Sans-serif body.
- **Predictable Components**: Subtle borders instead of shadows, standard corner radii, and clear focus states.

## Color Specifications
| Name | Value | Role |
| :--- | :--- | :--- |
| Background | `#F7F6F3` | Off-white base |
| Primary Text | `#111111` | High-contrast readability |
| Accent | `#8B0000` | Solid deep red for primary actions |
| Success | `#507D50` | Muted green for positive status |
| Warning | `#B58900` | Muted amber for alerts |

## Spacing Scale
- `8px`, `16px`, `24px`, `40px`, `64px`

## Component Guidelines
- **Buttons**: Primary (solid deep red), Secondary (outlined).
- **Cards**: Subtle border (`rgba(17, 17, 17, 0.1)`), no shadows.
- **Inputs**: Clean borders, accent color on focus.
- **Transitions**: 200ms `ease-in-out` constant.

## Layout Structure
1. **Top Bar**: Global navigation, progress, and status.
2. **Context Header**: Large serif headline, subtext, clear purpose.
3. **Workspace**: 2-column split (70/30) for focus and supplementary info.
4. **Proof Footer**: Checklist for development progress and verification.
